Another Parametricity Construction

I wanted to write down another very cute construction that seems to work thanks to parametricity.

Claim: For any category $\C$, there exists a type $B$, and for each object $C \in \C$ a type $T_C$ and a morphism $p_C : T_C \to B$, such that the functions $f$ that fit in the diagram \[\begin{CD} T_{C} @>{f}>> T_{D} \\ @VVV @VVV\\ B @= B \\ \end{CD}\] are in bijective correspondence to the morphisms $C \to D \in \C$.

In short, we can find a type $B$ such that we can faithfully embed $\C$ in types-over-$B$.

Here's how it works.

  1. We set $B$ to be the type of all presheaves over $\C$, i.e. functors $\C^\op \to \rset$.
  2. For each $C\in\C$, we set $T_C$ to be the type \[(b : B) \x b(C)\]
  3. Observe that asking for an $f$ that fits in the commutative diagram above is equivalent to asking for a term \[h: (b : B) \to b(C) \to b(D)\] Consider the parametricity free theorem for $h$. It tells us:
    Suppose we have presheaves $b_1, b_2 : B$, and a family of relations \[\_{\sim_C}\_ : b_1(C) \to b_2(C) \to \rset\] indexed by objects $C$ of $\C$. Suppose these relations are compatible with presheaf restriction in the sense that for any morphism $f : C \to D$ we have that \[x \sim_D y \Rightarrow x|_f \sim_C y|_f \] for any $x \in b_1(D), y\in b_2(D)$.

    For any $x_1 \in b_1(C)$ and $x_2 \in b_2(C)$, if $x_1 \sim_C x_2$, then $h\ b_1\ x_1 \sim_D h\ b_2\ x_2$.
