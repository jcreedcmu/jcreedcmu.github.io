Multiples of $n$

Before trying to write that program, let's try to go through the argument for arbitrary $n$. We start with $\alpha(q)$, and compute from the sum \[\sum_{i=0}^{n-1} \alpha(\zeta^i q) = \sum_{i=0}^{n-1} \sum_{t=0}^\infty t (\zeta^i q)^t\] \[= \sum_{i=0}^{n-1} \sum_{t=0}^\infty t \zeta^{it} q^t\] \[=\sum_{t=0}^\infty t q^t \sum_{i=0}^{n-1} \zeta^{it} \] \[=\sum_{t=0}^\infty t q^t \left(\begin{cases}n & \hbox{if $t\equiv 0\pmod n$;} \\ 0 &\hbox{otherwise.}\end{cases} \right)\] \[=\sum_{t=0}^\infty n^2t q^{nt} \] \[=n^2 \alpha(q^n) \] Now we single out the summand where $i=0$: \[\alpha(q) + \sum_{i=1}^{n-1} \alpha(\zeta^i q) = n^2 \alpha(q^n)\] and expand the definition of $\alpha$ in all summands except for $i=0$: \[\alpha(q) + \sum_{i=1}^{n-1} \left(\zeta^i q\over (1-\zeta^i q)^2\right) = n^2 \alpha(q^n) \tag{\dag}\] Let's now define \[ P_{ab\ldots}(n) = { 1-q^n \over (1-\zeta^{a}q)(1-\zeta^{b}q)\cdots }\] We will drop the $(n)$ when it is clear from context. Now multiply both sides of $(\dag)$ by $P_0^2$, to obtain \[\alpha(q) P_0^2 + q \sum_{i=1}^{n-1} \zeta^i P_{0i}^2 = n^2 P_0^2 \alpha(q^n) \tag{\ddag}\] Both $P_0^2$ and $-\sum_{i=1}^{n-1} \zeta^i P^2_{0i}$ turn out to be polynomials in $q$ with positive integer coefficients.

Lemma\[P_i = \sum_{j=0}^{n-1} q^j \zeta^{ij} \]

Proof: \[\left(\sum_{j=0}^{n-1} q^j \zeta^{ij}\right)(1-\zeta^i q) = \left(\sum_{j=0}^{n-1} q^j \zeta^{ij}\right) - \left(\sum_{j=0}^{n-1} q^{j+1} \zeta^{i(j+1)}\right)\] \[= \left(\sum_{j=0}^{n-1} q^j \zeta^{ij}\right) - \left(\sum_{j=1}^{n} q^j \zeta^{ij}\right) = 1 - q^n\zeta^{in} = 1- q^n\] ∎

Lemma \[ \sum_{j=0}^{n-1} \zeta^j P_j^2 = n^2 q^{n-1}\]

Proof: \[ \sum_{j=0}^{n-1} \zeta^j P_j^2 = \sum_{j=0}^{n-1} \zeta^j \left(\sum_{k=0}^{n-1} q^k \zeta^{jk} \right)^2\] \[= \sum_{j=0}^{n-1} \zeta^j \sum_{k_1=0}^{n-1} \sum_{k_2=0}^{n-1} q^{k_1+k_2} \zeta^{j(k_1 + k_2)} \] \[= \sum_{j=0}^{n-1} \zeta^j \left( \left(\sum_{p=0}^{n-2} (p+1)q^p\zeta^{jp} \right) + nq^{n-1}\zeta^{j(n-1)} + \left(\sum_{p=n}^{2n-2} (2n-p-1)q^p \zeta^{jp}\right) \right)\] \[= \left(\sum_{j=0}^{n-1} \sum_{p=0}^{n-2} (p+1)q^p\zeta^{j(p+1)} \right) + n^2q^{n-1} + \left(\sum_{j=0}^{n-1}\sum_{p=n}^{2n-2} (2n-p-1)q^p \zeta^{j(p+1)} \right)\] \[= \left( \sum_{p=0}^{n-2} (p+1)q^p\sum_{j=0}^{n-1} \zeta^{j(p+1)} \right) + n^2q^{n-1} + \left(\sum_{p=n}^{2n-2} (2n-p-1)q^p\sum_{j=0}^{n-1} \zeta^{j(p+1)} \right)\] \[ = n^2q^{n-1}\] ∎

Lemma \[\sum_{j=1}^{n-1} \zeta^j P^2_{0j} = {- P_0^2 + n^2q^{n-1} \over (1-q)^2} \]

Proof: \[\sum_{j=1}^{n-1} \zeta^j P^2_j =\sum_{j=1}^{n-1} \zeta^i P_{0j}^2 = {\sum_{j=1}^{n-1} \zeta^j P_j^2 \over (1-q)^2} = {- P_0^2 + \sum_{j=0}^{n-1} \zeta^j P_j^2 \over (1-q)^2} = {- P_0^2 + n^2q^{n-1} \over (1-q)^2}\] ∎

Recap

We've proven enough to show that $(\ddag)$ above, namely \[\alpha(q) P_0^2 + q \sum_{i=1}^{n-1} \zeta^i P_{0i}^2 = n^2 P_0^2 \alpha(q^n) \] is the same as \[\alpha(q) P_0^2 = q\left( P_0^2 - n^2q^{n-1} \over (1-q)^2\right) + n^2 P_0^2 \alpha(q^n) \]

Since $P_0 = \sum_{j=0}^{n-1} q^j$, we've eliminated all $\zeta$, but we still need to check that we actually get positive integer coefficients when we divide by $(1-q)^2$ in order to even conjecture that there is an implementable bijection between types. Let's define \[M_n = { P_0^2(n) - n^2q^{n-1} \over (1-q)^2}\] and inspect a few small examples, and their consecutive differences:

$n$\(M_n\)\(M_n - M_{n-1}\)
$1$$0$
$2$$1$$1$
$3$$1 + 4q + q^2$ $4q + q^2$
$4$$1 + 4q + 10q^2 + 4q^3 + q^4$ $9q^2 + 4q^3 + q^4$
$5$$1 + 4q + 10q^2 + 20q^3 + 10q^4 + 4q^5 + q^6$ $16q^3 + 9q^4 + 4q^5 + q^6$

Looks like a clear pattern. It's not too hard to see that if we define \[D_k = \sum_{i=0}^{k} (i+1)^2 q^{2k - i}\] we can see

Lemma \(M_{n+2} - M_{n+1} = D_n\)

Proof: Exercise. ∎

Corollary \(M_n = \sum_{k = 0}^{n-2} D_k\)

Therefore we make the conjecture that

Conjecture For all $n$, there is a reasonably simple functional program realizing the bijection \[\alpha(q) P_0^2(n) = q\left( \sum_{k = 0}^{n-2} D_k \right) + n^2 P_0^2(n) \alpha(q^n) \]
